# Icon Generation

This directory should contain PWA icons at the following sizes:
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these icons from your logo using tools like:
- https://app-manifest.firebaseapp.com/
- https://www.pwabuilder.com/
- https://realfavicongenerator.net/

The icons are referenced in the manifest.json file.
